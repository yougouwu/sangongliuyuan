# 🏛️ 三宫六院 · OpenClaw Multi-Agent System

仿照 [edict](https://github.com/cft0808/edict) 项目，将「三省六部」改为「三宫六院」主题的 OpenClaw 多智能体系统。

## 架构

```
三宫（后三宫）：
├── 乾清宫 (政务处理) - 对应中书省
├── 交泰殿 (皇后权力) - 对应尚书省
└── 坤宁宫 (祭祀/典礼) - 对应门下省

六院（东西十二宫）：
├── 东六宫：景仁宫、承乾宫、钟粹宫、延禧宫、永和宫、景阳宫
└── 西六宫：永寿宫、翊坤宫、储秀宫、启祥宫、长春宫、咸福宫
```

## 等级体系

皇后 → 皇贵妃 → 贵妃 → 妃 → 嫔 → 贵人 → 常在 → 答应

## 安装

```bash
bash install.sh
```
